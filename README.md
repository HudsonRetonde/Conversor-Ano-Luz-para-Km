## Conversor-Ano-Luz-para-Km
Conversor de Ano Luz para Km em HTML, CSS e JavaScript

Sabendo que 1 ano luz = 300000 km/s .vatiação do tempo;
variação do tempo = 362 * 24 * 60 * 60 s;
logo, 1 ano luz = 9460800000000 km.
